Random.Next
int Random.Next(int minValue, int maxValue);
Egy olyan álvéletlen számot ad, ami nagyobb, vagy egyenlő, mint minValue és kisebb, mint maxValue.

Használat

Így kell használni
int i, j, k

Random r;
r = new Random();
        
i = r.Next(1, 11);
j = r.Next(5, 20);
k = r.Next(1, 7);
Egyszóval az alábbi két sort végre kell hajtani a programban, mielőtt használhatóvá válik ez a függvény. (Létre kell hozni egy Random objektumot.)
i = r.Next(1, 11);
j = r.Next(5, 20);
k = r.Next(1, 7);

Példák

Egyszerű dobókocka:
using System;

class Program
{
    static void Main()
    {
        int i;
        Random r;
        r = new Random();
        
        i = r.Next(1, 7);
        Console.WriteLine(""+i+"-t dobtál");
        Console.ReadKey(true);
    }
}
