char.ToLower

char.ToLower

static char ToLower(char);
Ha paraméterül egy olyan karaktert kap, ami egy nagybetű, akkor a megfelelő kisbetűvel tér vissza. Egyéb esetben változtatás nélkül adja vissza a paraméterül kapott karaktert.

Példák

using System;

class Program
{
    public static void Main(string[] args)
    {
        Console.Write("Üss le egy billentyűt, aztán üss entert: ");
        char ch = char.Parse( Console.ReadLine() );
        
        Console.WriteLine("Ez volt az: "+ch);
        Console.WriteLine("Kisbetűsítve: "+char.ToLower(ch));
        Console.WriteLine("Nagybetűsítve: "+char.ToUpper(ch));
        Console.ReadKey(true);
    }
}
