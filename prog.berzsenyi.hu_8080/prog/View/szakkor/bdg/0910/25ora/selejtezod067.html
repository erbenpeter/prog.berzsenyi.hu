Selejtező forduló

Google Code Jam Africa 2010
Egy verseny selejtező fordulóján P feladatot kellett megoldani; az i. feladatot Si versenyző oldotta meg hibátlanul. A továbbjutáshoz legalább C feladat hibátlan megoldása szükséges. Határozzuk meg ezek alapján, hogy legfeljebb hány versenyző juthatott tovább.

Bemenet

A bemenet első sora a tesztesetek T számát adja meg. A következő T sor egy-egy tesztesetet ír le: szóközökkel elválasztva a problémák P, majd a továbbjutáshoz szükséges hibátlan megoldások C számát, végül pedig P egész, problémánként a hibátlan megoldók száma: S1,...,SP.

Kimenet

Minden esethez egy sor, amiben megadjuk a teszteset számát: "Case #x: ", majd azt a számot, hogy legfeljebb hány továbbjutó lehetséges.

Példa

Bemenet Kimenet 
2
2 2 73 100
3 2 245 272 238

Case #1: 73
Case #2: 377

Méretek

1 ≤ T ≤ 100
1 ≤ C ≤ P

Kis bemenet

1 ≤ P ≤ 6
0 ≤ Si ≤ 1000

Nagy bemenet

1 ≤ P ≤ 60
0 ≤ Si ≤ 1017
