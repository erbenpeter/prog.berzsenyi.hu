9. óra

A programozás OKTV első fordulójának megbeszélése.

4. feladat

Az F(1..6) vektor elemei kezdetben 0 értékűek. Két eljárást (függvényt) definiálunk, melyek paraméterei 1 és 6 közötti egészek.

Első (2 változatban)

Függvény Egyik1(a,b)
    Ciklus amíg F(a) > 0
        a := F(a)
    Ciklus vége
    Ciklus amíg F(b) > 0
        b := F(b)
    Ciklus vége
    Egyik1 := (b = a)
Függvény vége
Függvény Egyik2(a,b)
    Belső(a,x); Belső(b,y)
    Egyik2 := (x = y)
Függvény vége

Eljárás Belső(x,y)
    Ha F(x) > 0 akkor
        Belső(F(x),y)
        F(x) := y
    különben
        y := x
    Elágazás vége
Eljárás vége

Második

Eljárás Másik(a,b)
    Ciklus amíg F(a) > 0
        a := F(a)
    Ciklus vége
    Ciklus amíg F(b) > 0
        b := F(b)
    Ciklus vége
    F(b) := a
Eljárás vége
