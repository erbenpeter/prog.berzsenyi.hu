Console.WriteLine

Console.WriteLine

Legegyszerűbb esetben kiírja azt a string-et, vagy más változót a konzolra, amit paraméterként kapott, majd új sort kezd. A paraméterül kapott string lehet több string, vagy egyéb változó összefűzése. Az összefűzést a + karakter jelöli.

Példa

using System;

class Program
{
    public static void Main(string[] args)
    {
        int i = 42;
        //a következő két sor ugyanazt írja ki
        Console.WriteLine("Hello World!");
        Console.WriteLine("Hello "+"World!");
        //a következő három sor ugyanazt írja ki
        Console.WriteLine("A válasz: 42");
        Console.WriteLine("A válasz: "+42);
        Console.WriteLine("A válasz: "+i);

        Console.ReadKey(true);
    }
}

Megjegyzés

Meghívható paraméter nélkül is, ekkor csak egy új sort kezd. Viszont a nyitó-csukó zárójelpár a hívás végéről ekkor sem hagyható el.
Console.WriteLine;        //rossz
Console.WriteLine();      //jó
