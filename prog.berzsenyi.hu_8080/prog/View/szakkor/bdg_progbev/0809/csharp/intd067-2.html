int.Parse

int.Parse

static int Parse(string s)
Egész számmá alakít egy stringet. Az elfogadott számformátum a rendszer beállításaitól függ. Magyar Windows esetén például tizedesvessző kell, angol esetén viszont tizedespont. (Ezt a programozó átállíthatja, de ezzel egyelőre nem foglalkozunk.)

Példák

Egyszerű használat

using System;

class Program
{
    public static void Main(string[] args)
    {
        int i;
        Console.Write("Adj meg egy egész számot: ");
        i = int.Parse( Console.ReadLine() );

        Console.WriteLine("A beírt szám: "+i);
        Console.ReadKey(true);
    }
}

(Kicsit) érdekesebb esetek

Ez a függvény valójában egy stringet alakít számmá, mindegy, hogy honnan is származott az.
using System;

class Program
{
    public static void Main(string[] args)
    {
        string s1, s2;
        int i, j;

        s1 = "42";
        s2 = Console.ReadLine();

        i = int.Parse(s1);
        j = int.Parse(s2);

        Console.WriteLine(i+j);
        Console.ReadKey(true);
    }
}
