SQL HAVING

SQL HAVING

Példák


   SELECT diák, AVG(jegy) AS "tanulmányi átlag"
   FROM napló
   GROUP BY diák
   HAVING AVG(jegy) >= 4.0
   ORDER BY 2 DESC
